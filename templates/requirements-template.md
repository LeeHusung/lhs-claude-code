## 2. 핵심 기능 (우선순위순)

각 기능에 우선순위를 표기합니다:
- **필수**: 데모에 반드시 포함. 미구현 시 데모 불가.
- **권장**: 있으면 좋지만 없어도 데모 가능.
- **선택**: 시간이 남으면 구현.

3. [기능명]: [기능 설명] — 우선순위: 권장
4. [기능명]: [기능 설명] — 우선순위: 선택
